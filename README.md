# Photography Website

## Author
Mwangi Michael

## Description
This is a photography website that showcases my passion for capturing moments and expressing black culture through images. The site includes a gallery of my favorite photos, information about the photographer, and a contact section.

## Project Setup Instructions
1. Clone the repository:
   git clone https://github.com/MW4NG1/Photography-Website.git

2. Open the folder in VS Code

3. Open index.html using Live Server

## Live Site
View the live website here:
